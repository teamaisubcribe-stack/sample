// 01 — 바이브 코딩이란?

느낌으로
만드는 코드

바이브 코딩(Vibe Coding)은 Andrej Karpathy가 제안한 개념으로, 개발자가 직접 코드를 작성하는 대신 AI에게 원하는 것을 자연어로 설명하고, AI가 코드를 생성하면 결과만 확인하는 방식입니다.


문법이 아닌 의도가 중심이 되는 개발. 더 이상 모든 것을 암기할 필요가 없습니다.

// 이렇게 말하세요 →
"로그인 폼 만들어줘,
유효성 검사 포함해서"

// AI가 만들어줍니다 →
const LoginForm = () => {
  const [email, setEmail] = useState('')
  return <Form onSubmit={handleSubmit} />
}
// 02 — 핵심 특징

왜 바이브
코딩인가

01 ⚡

초고속 개발

아이디어에서 프로토타입까지 걸리는 시간을 획기적으로 단축합니다. AI가 반복적인 코드를 순식간에 생성합니다.

02 🧠

의도 기반 개발

문법보다 목적이 중요합니다. "무엇을 만들고 싶은가"에 집중하고, "어떻게"는 AI에게 맡기세요.

03 🔄

빠른 반복

결과가 마음에 들지 않으면? 다시 말하면 됩니다. 수정-피드백-개선의 사이클이 몇 초 안에 돌아갑니다.

04 🌊

몰입 상태 유지

문법 오류나 API 레퍼런스로 흐름이 끊기지 않습니다. 창의적인 흐름을 유지한 채 계속 앞으로 나아갑니다.

05 🛠️

비개발자도 가능

코딩 경험이 없어도 괜찮습니다. 아이디어와 논리적 사고만 있으면 누구든 소프트웨어를 만들 수 있습니다.

06 🎯

AI와의 협업

AI는 도구가 아닌 페어 프로그래머입니다. 당신의 비전을 이해하고 함께 코드를 만들어 나갑니다.

원하는 기능을 자연어로 설명합니다. "다크모드 토글 버튼 만들어줘", "MongoDB에서 유저 데이터 가져오는 API 만들어줘" — 당신의 언어로 말하면 됩니다.

🤖 AI가 생성

AI가 당신의 의도를 파악하고 코드를 작성합니다. 최적화된 구조, 에러 처리, 베스트 프랙티스까지 자동으로 적용됩니다.

✅ 확인 & 수정

결과를 확인하고 "더 간단하게", "성능 최적화해줘", "주석 추가해줘" 같은 피드백으로 바로 개선합니다.
